Git svör.
1.
cd = cd skipunin er notu til þess að breyta núverandi skrá.
cd ~ = þetta merkir bara heimasvæðið.
ls = Listi upplýsingar um valda möppu.
pwd = sýnir allt nafn valdar leiðar dæmi "H:\git_verkefni".
mkdir = ef leið tilgreindrar skráar er ekki til mun mkdir búa hana til.
2.
git clone = klónar geymslu í nýja skrá.
git log = skipunin tekur valkosti sem gilda um git Rev-lista stjórn til að stjórna hvað birtist og hvernig.
git status = Sýnir leiðir sem hafa mismunandi milli vístiölu skrá og núverandi yfirmaður fremja, slóðir sem hafa muninn vinnaandi tré og upphafsíðu skrá, og keupur æu vubby træe sem eru ekki fylgst með.
git diff = sýnir breyting á milli vinnandi tré og vístölu.
git checkout = upfærir skjöl í virkum trjám til að þær matchi við rétt verson í indexnum.
3.
git diff = sýnir breytingu á milli vinnandi trjáa og vístölu.
git diff --staged= er í rauninni bara stitting á --cached sem breytir næstu færslu hlutfallselga við nafnið og af því þetta er cached þýðir það að það sé vefsíða eða app.
git diff commit1 comit2 = sýnir tvö skjöl og muninn á milli þeirra.
